# 소수 찾기

## 문제

주어진 수 N개 중에서 소수가 몇 개인지 찾아서 출력하는 프로그램을 작성하시오.

## 입력

첫 줄에 수의 개수 N이 주어진다. N은 100이하이다. 다음으로 N개의 수가 주어지는데 수는 1,000 이하의 자연수이다.

## 출력

한 줄에 하나씩, 증가하는 순서대로 소수를 출력한다.

## 예제 입력

```
4
1 3 5 7
```

## 예제 출력

```
3
```

---

## ✏️ 문제 풀이

내가 접근한 방법

1. 10 미만의 자연수 중 합성수(↔소수) 구하기
2. 1에서 구한 수의 배수들을 제외하기
